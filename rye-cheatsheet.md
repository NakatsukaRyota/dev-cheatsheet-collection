## プロジェクト操作
### 既存プロジェクトを Rye 管理下に置く
```
rye init
```

### Pythonのバージョンを指定
```
rye pin 3.12
```

### 設定ファイルと依存関係を同期
```
rye sync
```
---

## パッケージ管理
### パッケージのインストール
```
rye add pyperclip
```

### 開発用パッケージのインストール
```
rye add --dev ruff
```

### パッケージのアンインストール
```
rye remove flask
```
---
## 仮想環境の実行
### 対話環境（Python REPL）を仮想環境で実行
```
rye run python --version
```

### スクリプトを仮想環境で実行
```
rye run script.py
```
---
## グローバルツール管理
### グローバルツールをインストール
```
rye install mypy
```

### グローバルツールのアンインストール
```
rye uninstall mypy
```
