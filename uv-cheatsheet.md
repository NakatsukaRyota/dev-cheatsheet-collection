## プロジェクトをスタートする
```bash
uv init hello-world
cd hello-world
```

## Pythonの特定のバージョンを指定する（pin）
プロジェクトで使用するPythonバージョンを固定
```bash
uv python pin 3.11
```

## 特定のバージョンのPythonをインストールする
```bash
uv python install 3.11
```

## インストールされているPythonのバージョンを確認する
```bash
uv python list
```

## プロジェクトに使用するPythonのバージョンを変更する
```bash
uv python pin 3.13
```

## Pythonのバージョンを指定して仮想環境を作成する
```bash
uv venv --python 3.13.3
```


## パッケージをインストールする
```bash
uv add requests
```

## パッケージを開発用にインストールする
```bash
uv add --dev ruff
```

## パッケージを削除する
```bash
uv remove requests
```

## インストールされたパッケージを確認する
```bash
uv pip list
```

## CLIツールをグローバルにインストールする
```bash
uv tool install ruff
```

## パッケージを同期する
'pyproject.toml'をもとにパッケージをインストールする<br>
通常はuv runを実行すると自動で実行される
```bash
uv sync
```

## スクリプトを実行する
現在のプロジェクトに定義された依存関係のもとでスクリプトを実行する
```bash
uv run your_script.py
```

## requirements.txtを出力する
```bash
uv pip freeze > requirements.txt
```

## ロックファイルを明示的に生成
```bash
uv lock
```

## 依存関係の構造をツリー形式で表示する
```bash
uv tree
```
